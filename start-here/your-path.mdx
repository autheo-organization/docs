## What is Autheo?

Autheo Chain is an EVM-compatible Layer 1 blockchain built on the Cosmos SDK. It introduces an NFT-based license system that governs participation in the validator set — every validator and delegator must hold an on-chain license credential.

For a deeper orientation, continue with [Autheo OS model](/start-here/autheo-os-model) or browse the [Components map](/start-here/components-map).
