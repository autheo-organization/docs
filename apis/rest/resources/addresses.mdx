## Get all addresses

```
curl -X 'GET' \

## Get specific address

`/addresses/{address_hash}`

Returns info for the given address hash.

**Request**

```
curl -X 'GET' \
  'https://evm-explorer.autheo.com/api/v2/addresses/0x3BCc6a2f85c3D0870D2A14d31bBaC9C3F9839BbF' \

## Get counters for a specific address

`/addresses/{address_hash}/counters`

Returns any counters for the given address.

**Request**

```
curl -X 'GET' \
  'https://evm-explorer.autheo.com/api/v2/addresses/0x3BCc6a2f85c3D0870D2A14d31bBaC9C3F9839BbF/counters' \
  -H 'accept: application/json'
```

**Response**

```
{
  "transactions_count": "0",
  "token_transfers_count": "0",
  "gas_usage_count": "0",
  "validations_count": "0"
}
```
